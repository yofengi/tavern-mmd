# 状态面板生成协议（模型侧 · 蓝灯常驻条目）

每次回复正文结束后，**另起一行**输出当前状态。**固态字段每轮必出**（即使没变也照原值重出）；**情境字段仅在对应情境那轮输出**（见下）。格式：

```
<g3>
hp=85/100;mood=害羞;loc=书房;npc=茜|好感:15,真红|好感:-20
</g3>
```

## 字段表

| 键名 | 含义 | 格式 | 示例 | 代谢 |
|---|---|---|---|---|
| `hp` | 生命 | 当前/最大 | `85/100` | 固态（继承） |
| `mood` | 心情 | 一个词或短语 | `害羞` | 固态（继承） |
| `loc` | 地点 | 地点名 | `书房` | 固态（继承） |
| `npc` | 在场角色 | 名|好感:数值,名2|好感:数值2 | `茜|好感:15,真红|好感:-20` | 固态（继承） |
| `fight` | 遭遇 | 敌名|HP:当前/最大,敌名2|HP:当前/最大2 | `野狼|HP:12/20` | 情境（用完即焚） |

## 铁律
- 用 `<g3>` 包裹，字段间用 `;` 分隔，键值用 `=`。
- **固态字段每轮必出**（panel 无跨气泡状态、靠每轮全量，缺了引擎才向历史兜底）。
- **情境字段 `fight`：只在该情境发生的那轮输出，情境一结束就停止输出**。不要每轮重复、不要在情境外保留——面板会随之消失（这是防高刺激信息污染上下文的代谢机制，不是 bug）。
- 值里**不要出现** `<` `>` `;` `=` 与换行（会破坏解析/HTML）。
- 列表字段条目用 `,` 分隔，名与属性用 `|`。
- 不要解释、不要加任何排版标记，只吐数据块。
