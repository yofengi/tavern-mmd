## 使用指引

### 选择方案
- **MMD 用户**：根据配色偏好选择日间版/夜间版（当前只有日间版）
- **本地酒馆用户**：本地酒馆无 MMD 的清污需求，可直接用纯 CSS 方案

### 导入 MMD
1. 打开 MMD 聊天页，点击「正则」或「正则替换」入口
2. 按方案文档的「使用方式」节，依次导入三条规则：
   - 规则1：全局美化 CSS
   - 规则2：清污引擎
   - 规则3：引号修复
3. 保存后刷新页面即可看到效果

### 自定义配色
所有方案都通过 CSS 变量驱动配色，换主题只需修改变量值，无需改 1000+ 行选择器。详见各方案文档的「配色变量」节。

## 模块化组件

| 组件 | 用途 | 文件 |
|---|---|---|
| **MMD 清污引擎** | MutationObserver 常驻监听，自动移除 MMD 原生日夜模式残留污染。所有 MMD 全局美化方案的通用基础设施。 | [mmd_cleanup_core.js](mmd_cleanup_core.js) |

清污引擎已集成进日间版方案的「规则2」中，也可单独使用或嵌入其他美化方案。
