# 文风控制

## 文风条目设计（写进卡的写作指导）
- 位置：本地酒馆 position=2（↑AT）；MMD平台同样写为世界书条目（蓝灯常驻）
- 内容三段：叙事视角与时态 / 正面规则（要什么）/ 负面规则（禁什么）
- 负面规则直接复用 character.md 八股化清单中适用项

## 文风条目模板
<style_guide>
视角: 第三人称限制视角（或第一/第二人称，与开场白一致）
节奏: [短句为主/长短交错/克制留白……选一种并一句话说明]
对话占比: [低/中/高]
描写侧重: [动作与环境/感官细节/对话潜台词……]
禁止: 模糊词（似乎/仿佛/宛如）、八股微表情（嘴角微微上扬/眼中闪过一丝）、语气声线描写、极端情绪词、"不是...而是..."句式、替用户发言或决定
</style_guide>

## 要点
- 文风模仿需求：让用户提供3段以上原文样本存入"资料/"，提炼为上述模板字段，不抄原文
- 文风条目≤300字：是约束清单不是散文
- 与开场白一致性：开场白本身必须符合文风条目，交付前互查
