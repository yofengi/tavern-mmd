## 如何使用

1. 选基调组（素雅/柔和/科技/考究/玩味/氛围）缩小范围。
2. 在该组里选具体风格（看明暗/MMD性能/适用场景）。
3. 到 palettes.md / layout-ui.md / fonts.md / decoration.md 取该 # 对应条目的 token。
4. 默认整套用；要混搭就跨文件取不同 # 的维度，混搭后按 ../style-system.md 第4节跑整体性检查；要单点改按第5节覆盖协议。
