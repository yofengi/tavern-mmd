# 布局 + UI 维度库（layout-ui）

本文件覆盖风格数据库的布局与 UI 两个维度。布局含密度/排列/圆角/分隔线；UI 含边框/阴影/状态/按钮形状。配合 ../style-system.md 与 palettes.md 使用。
